# Project 8: Retail Customer Analytics Dashboard

## Overview

Connected Google Looker Studio directly to the `retail_dw.dim_customers` dbt-transformed table in Google BigQuery.

## Key Metrics & Features

- **Total Customer Count:** Dynamic scorecard tracking total valid customers.
- **Customer Directory:** Interactive table displaying clean customer contact info.
- **Automated Refresh:** Automatically syncs whenever new records are transformed by the BigQuery / dbt pipeline.
